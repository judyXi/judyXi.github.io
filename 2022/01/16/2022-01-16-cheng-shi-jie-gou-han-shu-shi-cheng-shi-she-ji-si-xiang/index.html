函數式程式設計(思想)

  1. 定義：用一系列函數解決問題。

  • 函數可以賦值給變數，賦值後變數綁定函數。

  • 允許將函數作為參數傳入另一個函數。

  • 允許函數返回一個函數。

  1. 高階函數：將函數作為參數或返回值的函數。

函數作為參數

將核心邏輯傳入方法體，使該方法的適用性更廣，體現了物件導向的開閉原則。

lambda 運算式

  1. 定義：是一種匿名方法。
  2. 作用：作為參數傳遞時語法簡潔，優雅，代碼可讀性強。

隨時創建和銷毀，減少程式耦合度。

  1. 語法
  • 定義：

變數 = lambda 形參: 方法體

– 調用：

變數(實參)

  1. 說明：
  • 形參沒有可以不填
  • 方法體只能有一條語句，且不支援設定陳述式。

內置高階函數

  1. map（函數，可反覆運算物件）：使用可反覆運算物件中的每個元素調用函數，將返回值作為新可反覆運算物件元素；返回值為新可反覆運算對象。

  2. filter(函數，可反覆運算物件)：根據條件篩選可反覆運算物件中的元素，返回值為新可反覆運算物件。

  3. sorted(可反覆運算物件，key = 函數,reverse = bool值)：排序，返回值為排序結果。

  4. max(可反覆運算物件，key = 函數)：根據函數獲取可反覆運算物件的最大值。

  5. min(可反覆運算物件，key = 函數)：根據函數獲取可反覆運算物件的最小值。

"""
    內置高階函數
        這個函數的參數或者返回值是函數
    練習:exercise02
"""
class Wife:
    def __init__(self, name="", face_score=0, age=0, height=0):
        self.name = name
        self.face_score = face_score
        self.age = age
        self.height = height

    def __str__(self):
        return "%s-%d-%d-%d" % (self.name, self.face_score, self.age, self.height)

list_wife = [
    Wife("雙兒", 96, 22, 166),
    Wife("阿珂", 100, 23, 173),
    Wife("小郡主", 96, 22, 161),
    Wife("方怡", 86, 27, 166),
    Wife("蘇荃", 99, 31, 176),
    Wife("建寧", 93, 24, 163),
    Wife("曾柔", 88, 26, 170),
]

# 1.  map 映射
#  在老婆列表中獲取所有名稱
#  類似於：select
for element in map(lambda item: item.name, list_wife):
    print(element)

# 2. filter 篩檢程式
# 在老婆列表中獲取顏值大於90的所有老婆
#  類似於：find_all
for element in filter(lambda item: item.face_score > 90, list_wife):
    print(element)

# 3. max/min
# 在老婆列表中獲取顏值最高的老婆
#  類似於：get_max
print(max(list_wife,key = lambda item:item.face_score))
print(min(list_wife,key = lambda item:item.face_score))

# 4.sorted 排序
#   注意：沒有改變原有列表,而是返回新的
# 昇冪
for item in sorted(list_wife,key = lambda item:item.height):
    print(item)
# 降冪
for item in sorted(list_wife,key = lambda item:item.height,reverse=True):
    print(item)

函數作為返回值

邏輯連續，當內建函式被調用時，不脫離當前的邏輯。

閉包

  1. 三要素：

  • 必須有一個內嵌函數。

  • 內嵌函數必須引用外部函數中變數。

  • 外部函數返回值必須是內嵌函數。

  1. 語法
  • 定義：
def 外部函數名(參數):

    外部變數

    def 內建函式名(參數):

        使用外部變數

    return 內建函式名
  • 調用：

變數 = 外部函數名(參數)

變數(參數)

  1. 定義：在一個函數內部的函數,同時內建函式又引用了外部函數的變數。

  2. 本質：閉包是將內建函式和外部函數的執行環境綁定在一起的物件。

  3. 優點：內建函式可以使用外部變數。

  4. 缺點：外部變數一直存在於記憶體中，不會在調用結束後釋放，佔用記憶體。

  5. 作用：實現python裝飾器。

"""
    閉包
        三大要素：
            具有內外函數
            內建函式訪問外部函數變數
            外部函數返回內建函式
        字面意思：封閉程式運行環境  --> 保存
        體現：
            外部函數執行過後,棧幀不釋放,等待內建函式使用.
"""
def func01():
    a = 100
    def func02():
        print(a)
    return func02

# 調用外部函數,返回內建函式
result = func01()

# 因為python支援閉包,所以在外部函數執行過後,棧幀沒有釋放.
# 因此內建函式執行時,可以正確訪問外部嵌套變數.
result()

### 函數裝飾器decorator

1. 定義：在不改變原函數的調用以及內部代碼情況下，為其添加新功能的函數。
2. 語法

def 函數裝飾器名稱(func):

    def 內嵌函數(*args, **kwargs):

        需要添加的新功能

    return func(*args, **kwargs)

return內嵌函數

原函數 = 內嵌函數

@ 函數裝飾器名稱

def 原函數名稱(參數):

函數體

原函數(參數)

1. 本質：使用“@函數裝飾器名稱”修飾原函數，等同於創建與原函數名稱相同的變數，關聯內嵌函數；故調用原函數時執行內嵌函數。

原函數名稱 = 函數裝飾器名稱（原函數名稱）

1. 裝飾器鏈：

一個函數可以被多個裝飾器修飾，執行順序為從近到遠。

```python
"""
    裝飾器
"""

def func01():
    print("func01執行嘍")
    return "ok"

def func02(a):
    print(a, "func02執行嘍")

def print_func_name(func):
    def wrapper(*args, **kwargs):  # 合
        # 新功能
        print(func.__name__)
        # 舊功能
        return func(*args, **kwargs)  # 拆

    return wrapper

# func01 = 新功能  +  舊功能
# func01 = print_func_name + func01
func01 = print_func_name(func01)

print(func01())  # 執行的是內建函式

func02 = print_func_name(func02)

func02(100)

"""

def verify_permissions(func):
    def wrapper(*args, **kwargs):  # 合
        # 新功能

        # 舊功能

def print_execute_time(func):
    def wrapper(*args, **kwargs):
        start_time = time.time()
        result = func(*args, **kwargs)
        stop_time = time.time()
        print(stop_time - start_time)
        return result

    return wrapper

@print_execute_time
def func01():
    sum_value = 0
    for i in range(100):
        sum_value += i
    return sum_value

@print_execute_time
def func02(n):
    sum_value = 0
    for i in range(n):
        sum_value += i
    return sum_value

func01()

func02(100000)
