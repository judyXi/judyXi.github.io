生成器generator

  1. 定義：能夠動態(迴圈一次計算一次返回一次)提供資料的可反覆運算物件。
  2. 作用：在迴圈過程中，按照某種演算法推算資料，不必創建容器存儲完整的結果，從而節省記憶體空間。資料量越大，優勢越明顯。
  3. 以上作用也稱之為延遲操作或惰性操作，通俗的講就是在需要的時候才計算結果，而不是一次構建出所有結果。

生成器函數

  1. 定義：含有yield語句的函數，返回值為生成器對象。
  2. 語法
    • 創建：

def 函數名():

…

yield 數據

…

– 調用：

for 變數名 in 函數名():

語句

  1. 說明：
  • 調用生成器函數將返回一個生成器物件，不執行函數體。
  • yield翻譯為”產生”或”生成”
  1. 執行過程：
  • 調用生成器函數會自動創建反覆運算器物件。
  • 調用反覆運算器物件的__next__()方法時才執行生成器函數。
  • 每次執行到yield語句時返回資料，暫時離開。
  • 待下次調用__next__()方法時繼續從離開處繼續執行。
  1. 原理：生成反覆運算器物件的大致規則如下
  • 將yield關鍵字以前的代碼放在next方法中。
  • 將yield關鍵字後面的資料作為next方法的返回值。

返回單個結果用return

返回多個結果用yieid

"""
    反覆運算器 --> yield
"""

class SkillManager:
    def __init__(self):
        self.__skills = []

    def add_skill(self, skill):
        self.__skills.append(skill)

    def __iter__(self):
        # 你看見的代碼,不是這個樣子.
        # 代碼生成大致流程：
        # 1. 將yield以前的代碼,定義在__next__方法中
        # 2. 將yield以後的資料,作為__next__方法的返回值
        # print("準備")
        # yield self.__skills[0]
        # print("準備")
        # yield self.__skills[1]
        # print("準備")
        # yield self.__skills[2]

        for item in self.__skills:
            print("準備")
            yield item

manager = SkillManager()
manager.add_skill("九陽神功")
manager.add_skill("乾大挪移")
manager.add_skill("太極坤")

for item in manager:
    print(item)

def my_range(stop):
    number = 0
    while number < stop:
        yield number
        number += 1

my = my_range(5)

        item = iterator.__next__()

"""
    定義函數,在清單中獲取所有偶數.
        -- 傳統思想：將結果存入新列表再返回
        -- 生成器思想：將結果交給生成器對象推算

list01 = [43, 42, 68, 66, 78, 87, 453, 4]

def get_even01():
    list_result = []
    for item in list01:
        if item % 2 == 0:
            list_result.append(item)
		#返回單個結果用return
    return list_result

result = get_even01()
for item in result:
    print(item)

def get_even02():
    for item in list01:
        if item % 2 == 0:
		#返回多個結果用yieid
           yield item

result = get_even02()

    print(item)

內置生成器

枚舉函數enumerate

  • 語法：

for 變數 in enumerate(可反覆運算物件):

語句

for 索引, 元素in enumerate(可反覆運算物件):

語句

  • 作用：遍歷可反覆運算物件時，可以將索引與元素組合為一個元組。
"""
    參照下列代碼,自訂生成器my_enumerate.
"""

def my_enumerate(iterable):
    index = 0
    for item in iterable:
        yield (index, item)
        index += 1

dict01 = {"悟空": 26, "八戒": 28}
for item in my_enumerate(dict01):
    print(item)

for item in enumerate(dict01):
    print(item)

zip

  • 語法：

for item in zip(可反覆運算物件1, 可反覆運算對象2….):

語句

  • 作用：將多個可反覆運算物件中對應的元素組合成一個個元組，生成的元組個數由最小的可反覆運算物件決定。
"""
    參照下列代碼,自訂生成器my_zip
"""

def my_zip(iterable01, iterable02):
		#此語句判定誰最小，所以不會報錯
    for i in range(len(iterable01)):
        yield (iterable01[i], iterable02[i])

list01 = ["八戒", "悟空", "蘇大強"]
list02 = [102, 105]

for item in my_zip(list01, list02):
    print(item)

生成器運算式(表達式)

  1. 定義：用推導式形式創建生成器物件。
  2. 語法：變數 = ( 運算式 for 變數 in 可反覆運算物件 [if 真值表達式] )
"""
    生成器運算式
"""

# 列表推導式
list01 = [34, 43, 54, 65, 67, 7]
list02 = [item for item in list01 if item > 10]
for item in list02:
    print(item)

#元祖沒有推倒式，所以小括號給到生成器
generator02 = (item for item in list01 if item > 10)
for item in generator02:
    print(item)

for item in (item for item in list01 if item > 10):
    print(item)
"""
    1. 使用生成器運算式在list01中找出所有字串
    2. 使用生成器運算式找出所有大於10的整數
"""
list01 = ["悟空",57,"八戒",True,"三藏",5.8,10,90]

for element in (item for item in list01 if type(item) == str):
    print(element)

for element in (item for item in list01 if type(item) == int and item > 10):
    print(element)

【程式結構】_反覆運算( iterable / iterator )
每一次對過程的重複稱為一次“反覆運算”，而每一次反覆運算得到的結果會作為下一次反覆運算的初始值。例如：迴圈獲取容器中的元素。 可反覆運算對象iterable 定義：具有__iter__函數的物件，可以返回反覆運算器物件。
