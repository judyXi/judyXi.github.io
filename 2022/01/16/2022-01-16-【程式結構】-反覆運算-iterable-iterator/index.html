【程式結構】_反覆運算( iterable / iterator )

發表於 2022-01-16 更新於 2022-06-16 分類於 python 閱讀次數：

每一次對過程的重複稱為一次“反覆運算”，而每一次反覆運算得到的結果會作為下一次反覆運算的初始值。例如：迴圈獲取容器中的元素。

可反覆運算對象iterable

  1. 定義：具有__iter__函數的物件，可以返回反覆運算器物件。
  2. 語法

  • 創建：
    class 可反覆運算對象名稱:
    def iter(self):
    return 反覆運算器

  • 使用：
    for 變數名 in 可反覆運算對象:
    語句

  1. 原理：
    反覆運算器 = 可反覆運算對象.iter()

print(反覆運算器.next())

except StopIteration:

break
