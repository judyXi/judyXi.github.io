【程式結構】_包package

發表於 2022-01-05 更新於 2022-06-16 分類於 python 閱讀次數：

定義

將模組以資料夾的形式進行分組管理。

作用

讓一些相關的模組組織在一起，使邏輯結構更加清晰。

導入

import  包名.模組名 [as 模組新名]

import  包名.子包名.模組名 [as 模組新名]

from 包名 import 模組名 [as 模組新名]

from 包名.子包名 import 模組名 [as 模組新名]

from 包名.子包名.模組名 import 成員名 [as 屬性新名]

導入包內的所有子包和模組

from 包名 import *

from 包名.模組名 import *

init.py 文件

是包內必須存在的檔

會在包載入時被自動調用

all

記錄from 包 import * 語句需要導入的模組

案例：

根據下列結構，創建包與模組。

my_ project /

main.py

common/

init.py

list_helper.py

skill_system/

init.py

skill_deployer.py

skill_manager.py

  1. 在py中調用skill_manager.py中實例方法。
  2. 在py中調用skill_deployer.py中實例方法。
  3. 在py中調用list_helper.py中類方法。
