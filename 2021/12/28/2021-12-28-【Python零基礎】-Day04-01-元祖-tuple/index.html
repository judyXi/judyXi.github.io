【Python零基礎】_Day04_01_元祖 tuple

發表於 2021-12-28 更新於 2022-06-16 分類於 python 閱讀次數：

定義

  1. 由一系列變數組成的不可變序列容器。
  2. 不可變是指一但創建，不可以再添加/刪除/修改元素。

基礎操作

  • 創建空元組：
    元組名 = ()
    元組名 = tuple()

  • 創建非空元組：
    元組名 = (20,)
    元組名 = (1, 2, 3)
    元組名 = 100,200,300
    元組名 = tuple(可反覆運算對象)

  • 獲取元素：
    變數 = 元組名[索引]
    變數 = 元組名[切片] # 賦值給變數的是切片所創建的新列表

  • 遍歷元組：

正向：

for 變數名 in 列表名:
變數名就是元素

反向：

for 索引名 in range(len(列表名)-1,-1,-1):
元組名[索引名]就是元素

作用

  1. 元組與清單都可以存儲一系列變數，由於清單會預留記憶體空間，所以可以增加元素。

  2. 元組會按需分配記憶體，所以如果變數數量固定，建議使用元組，因為佔用空間更小。

  3. 應用：

    • 變數交換的本質就是創建元組：x, y = （y, x ）
    • 格式化字串的本質就是創建元祖：”姓名:%s, 年齡:%d”  % (“tarena”, 15)
