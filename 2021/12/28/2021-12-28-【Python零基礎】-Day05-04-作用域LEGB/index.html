【Python零基礎】_Day05_04_作用域LEGB

發表於 2021-12-28 更新於 2022-06-16 分類於 python 閱讀次數：

作用域LEGB

  1. 作用域：變數起作用的範圍。
  2. Local局部作用域：函數內部。
  3. Enclosing 外部嵌套作用域 ：函數嵌套。
  4. Global全域作用域：模組(.py文件)內部。
  5. Builtin內置模組作用域：py文件。

變數名的查找規則

  1. 由內到外：L -> E -> G -> B
  2. 在訪問變數時，先查找本地變數，然後是包裹此函數外部的函數內部的變數，之後是全域變數，最後是內置變數。

區域變數(局部變量)

  1. 定義在函數內部的變數(形參也是區域變數)
  2. 只能在函數內部使用
  3. 調用函數時才被創建，函數結束後自動銷毀

全域變數(全局變量)

  1. 定義在函數外部,模組內部的變數。
  2. 在整個模組(py檔)範圍內訪問（但函數內不能將其直接賦值）。

global 語句

  • 作用：
    在函數內部修改全域變數。
    在函數內部定義全域變數(全域聲明)。

  • 語法：
    global 變數1, 變數2, …

  • 說明
    在函數內直接為全域變數賦值，視為創建新的區域變數。
    不能先聲明局部的變數，再用global聲明為全域變數。

nonlocal 語句

  • 作用：
    在內層函數修改外層嵌套函數內的變數

  • 語法
    nonlocal 變數名1,變數名2, …

  • 說明
    在被嵌套的內函數中進行使用
