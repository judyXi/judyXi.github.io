【物件導向Object Oriented】_04_三大特徵-繼承

學習筆記
python
發布日期:   2021-12-28
文章字數:   874
閱讀時長:   3 分

語法角度講

繼承方法

  • 代碼:

class 父類:
def 父類方法(self):

方法體
class 子類(父類)：
def 子類方法(self):

方法體
兒子 = 子類()
兒子.子類方法()
兒子.父類方法()

  • 說明：
    子類直接擁有父類的方法.

內置函數

isinstance(對象, 類型)
返回指定物件是否是某個類的物件。
issubclass(類型，類型)
返回指定類型是否屬於某個類型。

繼承資料

  • 代碼
    class 子類(父類):
    def init(self,參數列表):
    super().init(參數列表)
    self.自身執行個體變數 = 參數

  • 說明
    子類如果沒有構造函數，將自動執行父類的，但如果有構造函數將覆蓋父類的。此時必須通過super()函式呼叫父類的構造函數，以確保父類執行個體變數被正常創建。

定義

重用現有類的功能，並在此基礎上進行擴展。
說明：子類直接具有父類的成員（共性），還可以擴展新功能。

優點

一種代碼複用的方式。

缺點

耦合度高：父類的變化，直接影響子類。

設計角度講

定義

將相關類的共性進行抽象，統一概念，隔離變化。

適用性

多個類在概念上是一致的，且需要進行統一的處理。

相關概念

父類（基類、超類）、子類（派生類）。
父類相對于子類更抽象，範圍更寬泛；子類相對于父類更具體，範圍更狹小。
單繼承：父類只有一個（例如 Java，C#）。
多繼承：父類有多個（例如C++，Python）。
Object類：任何類都直接或間接繼承自 object 類。

多繼承

一個子類繼承兩個或兩個以上的基類，父類中的屬性和方法同時被子類繼承下來。
同名方法的解析順序（MRO， Method Resolution Order）:
類自身 –> 父類繼承列表（由左至右）–> 再上層父類

   A

  / \

 /   \

B     C

 \   /

  \ /

   D
"""
    繼承 - 行為
        財產：錢不用孩子掙,但是可以花.
        皇位：江山不用太子打,但是可以坐.
        代碼：子類不用寫,但是可以直接用.
"""

class Person:
    def say(self):
        print("說話")

class Student(Person):
    def study(self):
        print("學習")
        super().say()

class Teacher(Person):
    def teach(self):
        print("教學")

p01 = Person()
# 父類物件,只能訪問父類成員
p01.say()

s01 = Student()
# 子類物件,可以訪問父類成員和本類成員
s01.say()
s01.study()

t01 = Teacher()
# 不能訪問兄弟類成員
# t01.study()

# 內置函數
# isinstance   物件  是一種  類型
# 人物件 是一種 人類型
print(isinstance(p01, Person))  # True
# 學生物件 是一種 人類型
print(isinstance(s01, Person))  # True
# 老師對象 是一種 學生類型
print(isinstance(t01, Student))  # False
# 人物件 是一種 老師類型
print(isinstance(p01, Teacher))  # False

# issubclass   類型  是一種  類型
# 人類型 是一種 人類型
print(issubclass(Person, Person))  # True
# # 學生類型 是一種 人類型
print(issubclass(Student, Person))  # True
# # 老師類型 是一種 學生類型
print(issubclass(Teacher, Student))  # False
# # 人類型 是一種 老師類型
print(issubclass(Person, Teacher))  # False

# type 對象  是  類型
# 人對象 是 人類型
print(type(p01) == Person)  # True
# 學生對象 是 人類型
print(type(s01) == Person)  # False
# 老師對象 是 學生類型
print(type(t01) == Student)  # False
# 人對象 是 老師類型
print(type(p01) == Teacher)  # False
"""
    繼承 - 資料
"""

class Person:
    def __init__(self, name=""):
        self.name = name

class Student(Person):
    def __init__(self, name="", score=0):
        super().__init__(name)
        self.score = score

# 如果子類沒有構造函數,直接使用父類構造函數
s01 = Student("悟空", 100)
print(s01.name)
