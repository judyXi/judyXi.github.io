【Python零基礎】_Day05_02_實參傳遞方式 argument

發表於 2021-12-28 更新於 2022-06-16 分類於 python 閱讀次數：

位置傳參

定義：實參與形參的位置依次對應。

  • fun01(1, 2, 3)

序列傳參

定義：實參用*將序列拆解後與形參的位置依次對應。

  • list01 = [4, 5, 6]
  • fun01(*list01)

關鍵字傳參

定義：實參根據形參的名字進行對應。

  • fun01(a=1, c=3, b=2)

字典關鍵字傳參

  1. 定義：實參用**將字典拆解後與形參的名字進行對應。
  2. 作用：配合形參的缺省參數，可以使調用者隨意傳參。
  • dict01 = {“c”: 3, “b”: 2, “a”: 1}
  • fun01(**dict01)
