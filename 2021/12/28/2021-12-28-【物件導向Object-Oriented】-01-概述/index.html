【物件導向Object Oriented】_01_概述

發表於 2021-12-28 更新於 2022-06-16 分類於 python 閱讀次數：

面向過程

  • 分析出解決問題的步驟，然後逐步實現。

例如：婚禮籌辦

  • 發請柬（選照片、措詞、製作）

  • 宴席（場地、找廚師、準備桌椅餐具、計畫菜品、購買食材）

  • 婚禮儀式（定婚禮儀式流程、請主持人）

  1. 公式：程式 = 演算法 + 資料結構

  2. 優點：所有環節、細節自己掌控。

  3. 缺點：考慮所有細節，工作量大。

物件導向

  • 找出解決問題的人，然後分配職責。

例如：婚禮籌辦

  • 發請柬：找攝影公司（拍照片、製作請柬）

  • 宴席：找酒店（告訴對方標準、數量、挑選菜品）

  • 婚禮儀式：找婚慶公司（對方提供司儀、制定流程、提供設備、説明執行）

  1. 公式：程式 = 物件 + 交互

  2. 優點

  • 思想層面：

  • 可模擬現實情景，更接近于人類思維。

  • 有利於梳理歸納、分析解決問題。

  • 技術層面：

  • 高複用：對重複的代碼進行封裝，提高開發效率。

  • 高擴展：增加新的功能，不修改以前的代碼。

  • 高維護：代碼可讀性好，邏輯清晰，結構規整。

  1. 缺點：學習曲線陡峭。

類和對象

基礎概念

  1. 抽象：從具體事物中抽離出共性、本質，捨棄個別、非本質過程。
  2. 類：一個抽象的概念，即生活中的”類別”。
  3. 對象：類的具體實例，即歸屬於某個類別的”個體”。
  4. 類是創建物件的”範本”。
  • 資料成員：名詞類型的狀態。
  • 方法成員：動詞類型的行為。

語法/定義類

  1. 代碼
    class 類名:
    “””文檔說明”””
    def init(self,參數列表):
    self.執行個體變數 = 參數

方法成員

  1. 說明
  • 類名所有單詞首字母大寫.
  • init 也叫構造函數，創建物件時被調用，也可以省略。
  •  self 變數綁定的是被創建的物件，名稱可以隨意。

創建對象(產生實體)

變數 = 構造函數 (參數清單)

實例成員

執行個體變數

  1. 語法
  • 定義：對象.變數名
  • 調用：對象.變數名
  1. 說明

  • 首次通過對象賦值為創建，再次賦值為修改.
    w01 = Wife()
    w01.name = “建寧”
    w01.name = “建甯公主”

  • 通常在構造函數(init)中創建。
    w01 = Wife(“建甯公主,24)
    print(w01.name)

  • 每個物件存儲一份，通過物件位址訪問。

  1. 作用：描述某個物件自己的資料。
  2. __dict__：物件的屬性，用於存儲自身執行個體變數的字典。

實例方法

  1. 語法
    (1) 定義：  def 方法名稱(self, 參數列表):
    方法體
    (2) 調用： 物件位址.實例方法名(參數清單)
    不建議通過類名訪問實例方法

  2. 說明
    (1) 至少有一個形參，第一個參數綁定調用這個方法的物件,一般命名為”self”。
    (2) 無論創建多少物件，方法只有一份，並且被所有物件共用。

  3. 作用：表示對象行為。
