【Python零基礎】_Day05_03_形參定義方式 parameter

發表於 2021-12-28 更新於 2022-06-16 分類於 python 閱讀次數：

缺省參數

  • 語法：
    def 函數名(形參名1=默認實參1, 形參名2=默認實參2, …):
    函數體
  • 說明：
    缺省參數必須自右至左依次存在，如果一個參數有缺省參數，則其右側的所有參數都必須有缺省參數。

缺省參數可以有0個或多個，甚至全部都有缺省參數。

位置形參

語法：
def 函數名(形參名1, 形參名2, …):
函數體

星號元組形參

  • 語法：
    def 函數名(*元組形參名):
    函數體
    def fun03(*args):
    print(args)
    fun03(1, 2, 3)

  • 作用：
    收集多餘的位置傳參。

  • 說明：
    一般命名為’args’
    形參列表中最多只能有一個

命名關鍵字形參

  • 語法：
    def 函數名(*, 命名關鍵字形參1, 命名關鍵字形參2, …):
    函數體
    def 函數名(*args, 命名關鍵字形參1, 命名關鍵字形參2, …):             函數體
    def fun04(*args, a, b, c):
    print(args)
    fun04(1, 2, 3, 4, 5, a=1, b=2, c=3)

  • 作用：
    強制實參使用關鍵字傳參

雙星號字典形參

  • 語法：
    def 函數名(**字典形參名):
    函數體
    def fun06(**kwargs):
    print(kwargs)
    fun06(a = 1,b = 2)

  • 作用：
    收集多餘的關鍵字傳參

  • 說明:
    一般命名為’kwargs’
    形參列表中最多只能有一個

參數自左至右的順序

位置形參 –> 星號元組形參 –> 命名關鍵字形參 –> 雙星號字典形參

參數自左至右的順序

位置形參 –> 星號元組形參 –> 命名關鍵字形參 –> 雙星號字典形參
