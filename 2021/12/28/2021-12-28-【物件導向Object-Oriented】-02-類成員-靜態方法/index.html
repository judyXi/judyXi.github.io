【物件導向Object Oriented】_02_類成員 / 靜態方法

發表於 2021-12-28 更新於 2022-06-16 分類於 python 閱讀次數：

類變數

  1. 語法
  • 定義：在類中，方法外定義變數。
    class 類名:
    變數名 = 運算式
  • 調用：類名.變數名
    不建議通過物件訪問類變數
  1. 說明
    (1) 存儲在類中。
    (2) 只有一份，被所有物件共用。
  2. 作用：描述所有物件的共有資料。

類方法

  1. 語法
  • 定義：
    @classmethod
    def 方法名稱(cls,參數列表):

方法體

  • 調用：類名.方法名(參數列表)
    不建議通過物件訪問類方法

  1. 說明
    (1) 至少有一個形參，第一個形參用於綁定類，一般命名為’cls’
    (2) 使用@classmethod修飾的目的是調用類方法時可以隱式傳遞類。
    (3) 類方法中不能訪問實例成員，實例方法中可以訪問類成員。

  2. 作用：操作類變數。

靜態方法

  1. 語法
  • 定義：
    @staticmethod
    def 方法名稱(參數列表):

方法體

  • 調用：類名.方法名(參數列表)
    不建議通過物件訪問靜態方法

  1. 說明
    (1) 使用@ staticmethod修飾的目的是該方法不需要隱式傳參數。
    (2) 靜態方法不能訪問實例成員和類成員

  2. 作用：定義常用的工具函數。
