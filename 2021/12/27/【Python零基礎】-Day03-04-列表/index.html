【Python零基礎】_Day03_04_列表

發表於 2021-12-27 更新於 2022-06-16 分類於 python 閱讀次數：

定義

由一系列變數組成的可變序列容器。

基礎操作

創建列表：

  • 列表名 = []

  • 清單名 = list(可反覆運算對象)

添加元素：

  • 列表名.append(元素)

  • 列表.insert(索引，元素)

定位元素：

  • 清單名[索引] = 元素
  • 變數 = 清單名[索引]
  • 變數 = 清單名[切片] # 賦值給變數的是切片所創建的新列表
  • 列表名[切片] = 容器 # 右側必須是可反覆運算物件，左側切片沒有創建新列表。遍歷列表：

正向：

  • for 變數名 in 列表名:
  • 變數名就是元素

反向：

  • for 索引名 in range(len(列表名)-1,-1,-1):
  • 清單名[索引名]就是元素

刪除元素：

  • 列表名.remove(元素)
  • del 清單名[索引或切片]

實際操作

清單(列表)VS字串

  1. 清單和字串都是序列,元素之間有先後順序關係。
  2. 字串是不可變的序列,列表是可變的序列。
  3. 字串中每個元素只能存儲字元,而清單可以存儲任意類型。
  4. 清單和字串都是可反覆運算物件。
  5. 函數：
    • 將多個字串拼接為一個。
    • result = “連接子”.join(列表)
    • 將一個字串拆分為多個。
    • 列表 = “a-b-c-d”.split(“分隔符號”)

列表推導式

  • 定義：
    使用簡易方法，將可反覆運算物件轉換為清單。

  • 語法：
    變數 = [運算式 for 變數 in 可反覆運算物件]
    變數 = [運算式 for 變數 in 可反覆運算物件 if 條件]

  • 說明:
    如果if真值表達式的布林值為False,則可反覆運算物件生成的資料將被丟棄。

列表推導式嵌套

  • 語法：
    變數 = [運算式 for 變數1 in 可反覆運算物件1 for 變數2 in可反覆運算物件2]

  • 傳統寫法：
    result = []

    for r in  [“a”, “b”, “c”]:

    for c in [“A”, “B”, “C”]:

    result.append(r + c)

  • 推導式寫法：
    result = [r + c for r in list01 for c in list02]
