【Python零基礎】_Day03_03_容器通用操作

發表於 2021-12-27 更新於 2022-06-16 分類於 python 閱讀次數：

數學運算子

  1. +：用於拼接兩個容器
  2. +=：用原容器與右側容器拼接,並重新綁定變數
  3. *：重複生成容器元素
  4. *=：用原容器生成重複元素, 並重新綁定變數
  5. < <= > >= == !=：依次比較兩個容器中元素,一但不同則返回比較結果。

成員運算子

  • 語法：
    數據 in 序列
    數據 not in 序列

  • n作用：
    如果在指定的序列中找到值，返回bool類型。

索引index

  1. 作用：定位單個容器元素
  2. 語法：容器[整數]
  3. 說明：
    正向索引從0開始，第二個索引為1，最後一個為len(s)-1。
    反向索引從-1開始,-1代表最後一個,-2代表倒數第二個,以此類推,第一個是-len(s)。

範例:

  • 字元 : P y t h o n
  • 正向索引 0 1 2 3 4 5
  • 反向索引 -6 -5 -4 -3 -2 -1

切片slice

  • 作用：
    定位多個容器元素。

  • 語法：
    容器[(開始索引):(結束索引)(:(步長))]

  • 說明：
    小括弧()括起的部分代表可省略
    結束索引不包含該位置元素
    步長是切片每次獲取完當前元素後移動的偏移量

內建函數

  1. len(x) 返回序列的長度
  2. max(x) 返回序列的最大值元素
  3. min(x) 返回序列的最小值元素
  4. sum(x) 返回序列中所有元素的和(元素必須是數數值型別)

描述 :

\n 代表分行符號

\r 返回游標至首行
