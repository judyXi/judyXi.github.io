【Python零基礎】_Day02_01_算數運算

算術運算子

+加法
-減法
*乘法
/ 除法：結果為浮點數
// 地板除：除的結果去掉小數部分
% 求餘
** 冪運算
優先順序從高到低： ( ** —> * / % // —> + - )

凡是浮點數參與的運算,結果一定是浮點數

print(4.3 + 6)  # 加法
print(4.3 - 6)  # 減法
print(4 * 6)    # 乘法
print(10 / 2)   # 除法 [/ 返回值結果是浮點數]

print(13 / 5)   # 除法 - 取小數商
print(13 % 5)   # 取餘(取模)  - 取餘數
print(13 // 5)  # 地板除(向下取整)  - 取整數商
print(-13 // 5)  # 地板除(向下取整)  - 取整數商
print(13.9 // 2)  # 地板除(向下取整)  - 取整數商

print(2 ** 10)   # 求冪 - 2的10次方
print(12 * 10 ** 9)   # 求冪 - 12乘以10的9次方

print(True + 1)   # True對應的數值為1
print(False + 1)   # False對應的數值為0

練習

  1. 超市西瓜 13 元一個, 問，你拿100 元可以買幾個西瓜，找零多少?
  2. 獲取數字：324 百位、十位、個位上對應的數字
#第一題
print('個數', 100 // 13)
print('找零', 100 % 13)

#第二題
print('百位:', 324 // 100)
print('十位:', 324 % 100 // 10)
print('十位:', 324 // 10 % 10)
print('個位:', 324 % 10)

增強運算子

  • 功能 : 變數與其他資料運算的結果重新賦值給變數(節省記憶體)
  • y += x 等同於 y = y + x
  • y -= x 等同於 y = y - x
  • y *= x 等同於 y = y * x
  • y /= x 等同於 y = y / x
  • y //= x 等同於 y = y // x
  • y %= x 等同於 y = y % x
  • y **= x 等同於 y = y ** x
a = 10

# 需求:在a的基礎上,加2 (自增)  [其他程式設計語言: ++/--]
a += 2   # 等效於
# a = a + 2
print(a)

# 創建新的變數(佔用記憶體)
a1 = a + 2
print(a1)

a //= 6
print(a)

比較運算子

“< 小於”
“<= 小於等於”
“>= 大於”
“>= 大於等於”
“== 等於等於”
“!= 不等於”
返回布林類型的值
比較運算的數學表示方式:0 <= x <= 100

a = 10
b = 15

print(a > b)   # 10 > 15  不成立(False)
print(a < b)   # 10 < 15  成立(True)

print(a <= 15)  # 10 <= 15 [10 < 15 或者 10 == 15]  成立(True)

# 判斷值是否在某個區間內
age = 16
print(0 <= age <= 18)   # 鏈式比較

# == 與 !=
print(10 == 2 * 5)    # 等於
print(10 != 2 * 5)    # 不等於
print(10.0 == 2 * 5)  # 等於

布林(邏輯)運算子

  • 短路邏輯
    說明：一旦結果確定，後面的運算式將不再執行
    擴展：邏輯運算時，儘量將複雜的（耗時的）判斷放在最後面

與and

  • 表示並且的關係，一假俱假。(都得滿足條件，結論才滿足條件表達並且的關係)

示例:
True and True   # True
True and False  # False
False and True  # False
False and False # False

或or

  • 表示或者的關係，一真俱真。(一個滿足就行 表達或者的關係)

示例:
True or True              # True
True or False             # True
False or True             # True
False or False            # False

非 not

  • 表示取反

例如：
not True   # 返回False
not False  # 返回True

示例:

# and - 描述:並且
# or - 描述:或者
# not - 描述:不(非)

# 需求:有房有車有存款
house = input('請問有房嗎?')
money = int(input('請問您的資產:'))

print(house == '有' and money >= 1000000)

'''
and
    運算式1      運算式2       結果
     True        True        True
     True        False       False
     False       True        False
     False       False       False

  結論: 一假俱假
'''

house = input('請問有房嗎?')
money = int(input('請問您的資產:'))

print(house == '有' or money >= 1000000)

'''
or
    運算式1      運算式2       結果
     True        True        True
     True        False       True
     False       True        True
     False       False       False

  結論: 一真俱真
'''

需求: 資產不少於1000000
money = int(input('請問您的資產:'))

print(not money < 1000000)

'''
not   
       運算式     結果
       True      False
       False     True
       
  結論: 真亦假,假亦真
'''

# bool 函數
# bool(物件)    # 判斷物件的真假狀態
# 為假的情況:
#  0
#  0.0
#  False
#  None
#  ''/""
#  空容器

# 數值
print(bool(10), bool(20))
print(10 and 20)    # 20
print(10 and 0)     # 0
print(0 and 0.0)    # 0
print(0 and 10)     # 0
# and: 優先返回為假(如果運算式1為真,則返回運算式2,否則直接返回運算式1)

print(10 or 20)    # 10
print(10 or 0)     # 10
print(0 or 0.0)    # 0.0
print(0 or 10)     # 10
# or: 優先返回為真(如果運算式1為真,則直接返回運算式1,否則返回運算式2)

print(not 0)    # not bool(0) --> True
print(not 1)    # not bool(1) --> False

# and
# 運算式1(True) --> 運算式2(True)  --> 結果(True)

# or
# 運算式1(True) -->  結果(True)
# 運算式2(True) -->  結果(True)

# and
# print(input('請問有房嗎?') == '有' and
#       int(input('請問您的資產:')) >= 1000000)

# or
# print(input('請問有房嗎?') == '有' or
#       int(input('請問您的資產:')) >= 1000000)

# not
print(not input('請問有房嗎?') == '有')

短路運算

  • 一但結果確定，後面的語句將不再執行。

身份運算子

語法:

  • x is y
  • x is not y

作用：

  • is 用於判斷兩個物件是否是同一個物件,是時返回True,否則返回False。
  • is not 的作用與is相反

優先順序

高到低：

算數運算子 —> 比較運算子 —> 快捷運算子 —> 身份運算子 —> 邏輯運算子

語句

行

  1. 物理行：程式師編寫代碼的行。
  2. 邏輯行：python解譯器需要執行的指令。
  3. 建議一個邏輯行在一個物理行上。
  4. 如果一個物理行中使用多個邏輯行，需要使用分號；隔開。
  5. 如果邏輯行過長，可以使用隱式換行或顯式換行。
  • 隱式換行：所有括弧的內容換行,稱為隱式換行
  • 括弧包括:  ()  []   {} 三種
  • 顯式換行：通過折行符 \ (反斜線)換行，必須放在一行的末尾，目的是告訴解譯器,下一行也是本行的語句。

pass 語句

  • 通常用來填充語法空白。
# 3個物理行,3個邏輯行
a = 1
b = 2
print(a, b)

# 2個物理行,3個邏輯行(不推薦)
a = 1;b = 2
print(a, b)

year = -1900

# 4個物理行,1個邏輯行
# 顯式換行: \  續行符
res = year > 0 and \
      (year % 4 == 0 and
       year % 100 != 0 or
       year % 400 == 0)
    # 隱式換行

print(res)
