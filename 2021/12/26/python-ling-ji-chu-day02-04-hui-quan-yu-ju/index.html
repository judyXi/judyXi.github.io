while語句

  • 作用:
    可以讓一段代碼滿足條件，重複執行。
  • 語法:
    while 真值表達式: 
	循環體 
else: 
	語句塊

執行流程

  1. 先定義一個初始條件
  2. 先判斷真值表達式，測試布林值為True 或 False
  3. 如果真值表達式為True，則執行語句塊1，再返回到第一步，判斷真值表達式
  4. 如果真值表達式為False，則執行語句塊2，然後結束while迴圈語句的執行
  5. 當真值表達式為假，則執行else中的內容，else部分可以省略
content = input("請輸入內容：")
# if content != "":
#     print("真值")

if content:  # bool(content)
    print("真值")

# 根據一個條件，為變數進行賦值。
# if input("請輸入性別：") == "男":
#     sex = 1
# else:
#     sex = 0

sex = 1 if input("請輸入性別：") == "男" else 0

break語句

break語句
作用：用來終止當前迴圈語句的執行

說明：

  1. 當break語句執行後，此迴圈語句中break之後的語句將不再執行；
  2. break語句終止迴圈時，迴圈語句的else子句的語句將不會被執行；
  3. break語句只能終止當前迴圈，如果有迴圈嵌套時，終止本次迴圈，執行外層迴圈
  4. break語句只能在迴圈(while for語句)內部使用

continue 語句
作用：跳過本次迴圈，繼續下一次新的迴圈

說明：

  1. continue語句執行後將不再執行本次迴圈內continue之後的語句
  2. 在while語句中，執行continue將會直接跳轉到while語句的真值表達式處 重新判斷迴圈
  3. continue語句只能在迴圈(while for語句)內部使用

總結

  • 不同點 break語句：用來終止當前迴圈。
  • continue語句：跳出本次迴圈繼續下一次迴圈。

相同點 :

  1. break語句與continue語句都是使用在while、for迴圈中
  2. break語句與continue語句之後的語句不在執行

for 語句

  • 作用:
    用來遍歷可反覆運算物件的資料元素。
    可反覆運算物件是指能依次獲取資料元素的對象，例如：容器類型。

    for 變數清單 in 可反覆運算物件： 
	循環體 
else ： 
	語句塊

  • 說明:
    else子句可以省略。
    在循環體內用break終止迴圈時,else子句不執行。

可反覆運算物件 (可迭代對象)

指能依次獲取資料的物件，包括：
非空字串str、 非空列表list、 非空元組tuple、 非空字典dict、 非空集合set、 反覆運算器等。

執行流程

  1. 變數依次接收可反覆運算物件給出的元素，然後執行循環體；
  2. 可反覆運算物件不能提供資料元素後，執行else子句部分語句，然後退出迴圈；
  3. else子句部分可省略

range 函數

作用:

  • 用來創建一個生成一系列整數的可反覆運算物件(也叫整數序列生成器)。

語法:

range(開始點，結束點，間隔)

說明:

  • 函數返回的可反覆運算物件可以用for取出其中的元素
  • 返回的數位不包含結束點
  • 開始點默認為0
  • 間隔預設值為1

參數：

  1. start：起始值，默認為0，可以省略
  2. stop：終止值，必須存在，且終止值無法取得
  3. step：步長，默認為1，可正可負，可以省略
    • 返回值：一個整數的range物件
    • 說明：從start開始，每次生成一個整數後移動step，直到stop為止

import random

random_number = random.randint(1, 100)
print(random_number)
count = 0
while count < 5:
    count += 1
    input_number = int(input("請輸入數位："))
    if input_number > random_number:
        print("大了")
    elif input_number < random_number:
        print("小了")
    else:
        print("猜對了，總共猜了" + str(count) + "次")
        break
else:
    # 當迴圈不滿足條件時退出，才執行以下代碼
    print("失敗了")
# 第二題 : 列印數位矩形
# 輸入邊長,列印對應邊長的數字矩形
# 如: 邊長為4
#   0 1 2 3     [col<side+row]
#   1 2 3 4     [col<side+row]
#   2 3 4 5     [col<side+row]
#   3 4 5 6     [col<side+row]

'''
分析
                  起始條件      結束條件      控制因素
    外層迴圈(row)   row=0       row<side      行
    內層迴圈(col)   col=row     col<side+row  列 [難點]
'''

side = int(input('請輸入邊長:'))

row = 0
while row < side:
    col = row
    while col < side + row:
        print(col, end=' ')
        col += 1
    print()
    row += 1
