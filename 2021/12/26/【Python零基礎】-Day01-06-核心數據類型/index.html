【Python零基礎】_Day01_06_核心數據類型

發表於 2021-12-26 更新於 2022-06-16 分類於 python 閱讀次數：

核心資料類型

  1. 在python中變數沒有類型，但關聯的物件有類型。
  2. 通過type函數可查看。

空值對象 None

  1. 表示不存在的特殊物件。
  2. 作用：占位元和解除與對象的關聯。

整形int

  • 表示整數，包含正數、負數、0。
    如： -5, 100, 0
  1. 字面值：
    • 十進位：5
    • 二進位：0b開頭，後跟1或者1
    • 八進制：0o開頭，後跟0~7
    • 十六進位：0x開頭，後跟09,AF,a~f

浮點型float

  1. 表示小數，包含正數、負數，0)。
  2. 字面值：
    • 小數：1.0   2.5
    • 科學計數法：e/E (正負號) 指數
    • 1.23e-2 (等同於0.0123)
    • 1.23456e5(等同於123456.0)

字串str

是用來記錄文本資訊(文字資訊)。

  • 字面值：雙引號

複數complex

  • 由實部和虛部組成的數位。
  • 虛部是以j或J結尾。
  • 字面值： 1j     1+1j   1-1j
  • 語法格式：
    • 實部係數 + 虛部係數j / 實部係數 + 虛部係數J
    • 說明：實部係數可以沒有，虛部係數不能不存在，沒有可以用0表示

布林bool

  • 用來表示真和假的類型
  • True 表示真(條件滿足或成立)，本質是1
  • False 表示假(條件不滿足或不成立)，本質是0
  • True、False 關鍵字首字母大寫。
