練習題1

練習1：在終端中錄入一個半徑，輸出圓形的面積(3.14 × 半徑的平方)，輸出的格式：圓形面積是：xxx

練習2：在終端中分別錄入3個資料(分鐘數、小時數、天數)，輸出總秒數。

練習3：在終端中首先獲取一個變數，再獲取一個變數，然後交換他們，最後輸出兩個變數。

練習4：獲取用戶輸入的秒數，計算是多少小時多少分鐘多少秒？

解法:

# 練習1：在終端中錄入一個半徑，輸出圓形的面積(3.14 × 半徑的平方)，輸出的格式：圓形面積是：xxx
radius = input("請輸入圓形半徑:")
area = 3.14 * float(radius) ** 2
print("圓形面積是：" + str(area))
# 練習2：在終端中分別錄入3個資料(分鐘數、小時數、天數)，輸出總秒數
minute = float(input("請輸入分鐘："))
hour = float(input("請輸入小時："))
day = float(input("請輸入天："))

result = minute * 60 + hour * 60 * 60 + day * 24 * 60 * 60

print("總秒數是：" + str(result))
# 練習3：在終端中首先獲取一個變數，再獲取一個變數，然後交換他們，最後輸出兩個變數。
variable01 = input("請輸入第一個變數：")

# variable01= variable02

# 練習4：獲取用戶輸入的秒數，計算是多少小時多少分鐘多少秒？
seconds = int(input('請輸入秒數:'))

#minute = seconds // 60 % 60
minute = seconds % 3600 // 60
second = seconds % 60

print(str(hour) + '小時' + str(minute) + '分鐘' + str(second) + '秒')

result = hour * 3600 + minute * 60 + second
print('總秒數:' + str(result))

#  整除: 年份 % 4 == 0

res = year > 0 and \
      (year % 4 == 0 and

sums += number % 10

print('四位數的和為:', sums)
#練習題5:在終端中錄入距離，時間，初速度，計算加速度。
"""
    勻變速直線運動的位移與時間公式：
        距離=初速度 * 時間 + 加速度 * 時間平方的一半
    輸入：
        距離:100
        初速度:2
        時間:5
    輸出：加速度7.2

    公式：
        距離=初速度 * 時間 + 加速度 * 時間**2 * 0.5
        (距離 - 初速度 * 時間) * 2/ 時間**2 = 加速度
"""
distance = float(input("請輸入距離："))
initial_velocity = float(input("請輸入初速度："))
time = float(input("請輸入時間："))
accelerated_speed = (distance - initial_velocity * time) * 2 / time ** 2
print("加速度是：" + str(accelerated_speed))

if weight4 > 0 and weight3 > 0 and weight2 > 0 and weight1 > 0:

     max_weight = weight1

 else:

#練習題2:輸入一個成績，判斷該成績是屬於哪個成績等級？

# 隨機猜數：產生1個1到100之間的亂數：
#練習題3:玩家重複猜測，直到猜對為止。每次提示：大了、小了，若猜對了則列印恭喜猜對並結束猜數，列印總共猜了多少次。

'''
分析:
    1 產生一個1-100(包含)之間的隨機整數
    2 創建變數,記錄次數
    3 迴圈輸入數:
        記錄次數 += 1
        判斷輸入的數與亂數的大小關係:
            大: 輸入的數 > 亂數
            小: 輸入的數 < 亂數
            等: 輸入的數 == 亂數,列印猜測次數,結束迴圈
'''

import random

randNum = random.randint(1, 10)  # 隨機產生[1,10]的整數
# print(randNum)
times = 0  # 記錄次數

while True:  # 未知迴圈次數
    userNum = int(input('請輸入一個[1,100]的整數'))
    times += 1

    if userNum > randNum:
        print('大了')
    elif userNum < randNum:
        print('小了')
    else:  # userNum == randNum
        print('恭喜您猜對了', "共使用" + str(times) + "次機會")
        break
