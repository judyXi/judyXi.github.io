發表於 2021-12-26 更新於 2022-06-16 分類於 python 閱讀次數：

登錄驗證

第1次可以輸入用戶名與密碼（輸入的總次數不能超過3次）

如果用戶名與密碼錯誤，提示：用戶名用戶名錯誤
如果用戶名正確，密碼錯誤，不用輸入用戶名，驗證密碼
在3次機會之內，如果用戶名與密碼正確，則提示：登錄成功
每次輸入列印剩餘多少次機會？
如果3次機會用完，列印：帳號被鎖定，程式結束
