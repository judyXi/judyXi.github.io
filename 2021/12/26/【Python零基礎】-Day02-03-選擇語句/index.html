【Python零基礎】_Day02_03_選擇語句

發表於 2021-12-26 更新於 2022-06-16 分類於 python 閱讀次數：

If elif else 語句

作用:
根據條件有選擇性的執行某條語句或某些語句。

if結構

以下使用場景：適用於判斷事物1種情況

if-else結構

以下使用場景：適用於判斷事物2種情況

if-elif-else 結構

以下使用場景：適用於判斷事物3種或3種以上的情況

結構縮進：官方描述為4個空格（不同的編輯器縮進存在不同），表示代碼的結構層次。

  • 說明:
  • elif 子句可以有0個或多個。
  • else 子句可以有0個或1個，且只能放在if語句的最後。

if 語句的真值表達式

if 100:
print(“真值”)
等同於
if bool(100):
print(“真值”)

條件運算式

  • 語法：變數 = 結果1 if 條件 else 結果2。
  • 作用：根據條件(True/False) 來決定返回結果1還是結果2。
  • 執行： 先執行真值表達式，若為True，返回運算式1，否則，返回運算式2。
  • 使用場景： 適用於簡單事物的2種的情況

代碼調試

功能：讓程式中斷，逐行執行。
目的：審查代碼的執行流程，明確每行執行的結果
流程：
1添加中斷點：點擊代碼行【】前，出現紅點。
2調試模式：右擊選擇 “debug” 。
3點擊“向下”或F7，逐行語句執行。
4Ctrl + F2 結束調試模式。

練習題

第一題 : 先在終端中錄入一個數位

  • 再錄入一個運算子

  • 最後錄入一個數字

  • 列印計算的結果

  • 要求：如果運算子不是+ - * /提示“運算子輸入有誤”

第二題 : 在終端中錄入月份，然後列印天數。

  • 輸入：2 輸出：28天

  • 輸入：1 3 5 7 8 10 12 輸出：31天

  • 輸入：4 6 9 11 輸出： 30天
