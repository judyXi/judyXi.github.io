算術運算子

+加法
-減法
*乘法
/ 除法：結果為浮點數
// 地板除：除的結果去掉小數部分
% 求餘
** 冪運算
優先順序從高到低： ( ** —> * / % // —> + - )

凡是浮點數參與的運算,結果一定是浮點數

練習

  1. 超市西瓜 13 元一個, 問，你拿100 元可以買幾個西瓜，找零多少?
  2. 獲取數字：324 百位、十位、個位上對應的數字

增強運算子

  • 功能 : 變數與其他資料運算的結果重新賦值給變數(節省記憶體)
  • y += x 等同於 y = y + x
  • y -= x 等同於 y = y - x
  • y *= x 等同於 y = y * x
  • y /= x 等同於 y = y / x
  • y //= x 等同於 y = y // x
  • y %= x 等同於 y = y % x
  • y **= x 等同於 y = y ** x

比較運算子

“< 小於”
“<= 小於等於”
“>= 大於”
“>= 大於等於”
“== 等於等於”
“!= 不等於”
返回布林類型的值
比較運算的數學表示方式:0 <= x <= 100

布林(邏輯)運算子

  • 短路邏輯
    說明：一旦結果確定，後面的運算式將不再執行
    擴展：邏輯運算時，儘量將複雜的（耗時的）判斷放在最後面

與and

  • 表示並且的關係，一假俱假。(都得滿足條件，結論才滿足條件表達並且的關係)

示例:
True and True   # True
True and False  # False
False and True  # False
False and False # False

或or

  • 表示或者的關係，一真俱真。(一個滿足就行 表達或者的關係)

示例:
True or True              # True
True or False             # True
False or True             # True
False or False            # False

非 not

  • 表示取反

例如：
not True   # 返回False
not False  # 返回True

示例:

短路運算

  • 一但結果確定，後面的語句將不再執行。

身份運算子

語法:

  • x is y
  • x is not y

作用：

  • is 用於判斷兩個物件是否是同一個物件,是時返回True,否則返回False。
  • is not 的作用與is相反

優先順序

高到低：

算數運算子 —> 比較運算子 —> 快捷運算子 —> 身份運算子 —> 邏輯運算子

語句

行

  1. 物理行：程式師編寫代碼的行。
  2. 邏輯行：python解譯器需要執行的指令。
  3. 建議一個邏輯行在一個物理行上。
  4. 如果一個物理行中使用多個邏輯行，需要使用分號；隔開。
  5. 如果邏輯行過長，可以使用隱式換行或顯式換行。
  • 隱式換行：所有括弧的內容換行,稱為隱式換行
  • 括弧包括:  ()  []   {} 三種
  • 顯式換行：通過折行符 \ (反斜線)換行，必須放在一行的末尾，目的是告訴解譯器,下一行也是本行的語句。

pass 語句

  • 通常用來填充語法空白。
