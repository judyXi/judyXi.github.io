【Python零基礎】_Day01_06_核心數據類型

學習筆記 線上課程
python
發布日期:   2021-12-26
文章字數:   1.3k
閱讀時長:   5 分

核心資料類型

  1. 在python中變數沒有類型，但關聯的物件有類型。
  2. 通過type函數可查看。

空值對象 None

  1. 表示不存在的特殊物件。
  2. 作用：占位元和解除與對象的關聯。

整形int

  • 表示整數，包含正數、負數、0。
    如： -5, 100, 0
  1. 字面值：
    • 十進位：5
    • 二進位：0b開頭，後跟1或者1
    • 八進制：0o開頭，後跟0~7
    • 十六進位：0x開頭，後跟09,AF,a~f

浮點型float

  1. 表示小數，包含正數、負數，0)。
  2. 字面值：
    • 小數：1.0   2.5
    • 科學計數法：e/E (正負號) 指數
    • 1.23e-2 (等同於0.0123)
    • 1.23456e5(等同於123456.0)

字串str

是用來記錄文本資訊(文字資訊)。

  • 字面值：雙引號

複數complex

  • 由實部和虛部組成的數位。
  • 虛部是以j或J結尾。
  • 字面值： 1j     1+1j   1-1j
  • 語法格式：
    • 實部係數 + 虛部係數j / 實部係數 + 虛部係數J
    • 說明：實部係數可以沒有，虛部係數不能不存在，沒有可以用0表示
      範例 :
      3.4+5j
Out[7]: (3.4+5j)
4.6J
Out[8]: 4.6j
type(4.6J)
Out[9]: complex

布林bool

  • 用來表示真和假的類型
  • True 表示真(條件滿足或成立)，本質是1
  • False 表示假(條件不滿足或不成立)，本質是0
  • True、False 關鍵字首字母大寫。

總範例:


name = "lzmly"
# 1. None 空
# -- 占位：只希望有一個變數，但指向的物件還不能確定.
skill = None
# ...
skill = "乾坤大挪移"

# -- 解除與物件的綁定關係
name = None

# 2. 整形(整數)  int
# 字面值
# -- 十進位：0 1 2 3 ...9 10
number01 = -1
number01 = 1
number01 = 0
b = '10'
print(a, type(a))
print(b, type(b))

# -- 二進位：0 1 10  11  100  101 ..
number02 = 0b10
print(number02)

# -- 八進制：0 1 2 3 ... 7  10
number03 = 0o10
print(number03)

# -- 十六進位：0 --9  a(10) -- f(15)  10   11
number04 = 0x11
print(number04)

# 浮點數型 - float (描述小數)
b = 3.14     # 字面值
b = -3.14
b = 0.0
b = 23E10    # 極大數(科學計數法)  23乘以10的10次方
b = 23e-5    # 極小數(科學計數法)
print(b, type(b))

# 科學計數法
number06 = 5e3
print(number06)

number07 = 0.00000000000000000005
number07 = 500000000000000000000.0
print(number07)

# 4. 字串(文字)  str
name = "lzmly"
str_number = "1000"

# 布林類型 - bool (描述邏輯關係-成立/不成立)
# 取值: True(成立,數值為1)/False(不成立,數值為0)
d = 1 > 5
print(d, type(d))

number01 = 5
number02 = 8
print(number01 > number02)  # 5 > 8 False
print(number01 < number02)  # 5 < 8 True
print(number01 == number02)  # 5 == 8 False
# print(5 > "10")  # 不能用整數與字串進行大小的比較
print(5 == "5")  # False  類型不一樣

# 空值物件 - None  (描述特殊不存在的物件)
# 作用: 占位元或解除與物件的關聯關係
a = None        # 佔用
print(a)

a = 10
print(a)

a = None        # 解除
print(a)

# input函數
# 返回值的類型: 字串(str)
data = input('請輸入您的密碼:')
print('密碼是:', data, type(data))

# 數學運算
result = 1000 + 1
# 文字拼接
result = "1000" + "1"
print(result)

# 5. 資料類型轉換
# input函數的返回值永遠都是字串類型
str_number = input("請輸入一個數字：")

# str --> int
# print(type(變數)) --> 獲取變數所關聯的物件類型
# re = int(str_number)

# str --> float
re = float(str_number)
print(type(re))

# 注意：int()轉換時，資料必須像整數
# 注意：float()轉換時，資料必須像小數

# 其他類型 --> str
re = str(100.6)

序列賦值

  • 場景:資料交換
 
# 現有變數 a=30, b=60，通過程式實現變數值的交換：a=60,  b=30
a = 30
b = 60

# 普通方法
c = a
a = b
b = c
print(a, b)

# 序列賦值
b, a = a, b  # b, a = 30, 60
print(a, b)

# 注意:
# 變數名個數小於資料物件個數(不可以)
x, y = 10, 20, 30
print(x, y)

# 變數名個數大於資料物件個數(不可以)
x, y, z, q = 10, 20, 30
print(x, y)

序列賦值內存圖

資料類型轉換

  • 同類型的資料才可以運算
# input函數
# 返回值的類型: 字串(str)
data = input('請輸入您的密碼:')
print('密碼是:', data, type(data))

# str --> int    常用
a = '123'
b = int(a)   # 將a轉換為整型
print(b, type(b))

# str --> float   常用
c = '3.14'
d = float(c)
print(d, type(d))

# 場景:字串的拼接
# int --> str
e = str(123)
print(e, type(e))

# float --> str
f = str(3.14)
print(f, type(f))

# float --> int
s = int(23.94)    # 直接去除小數部分
print(s, type(s))

# 注意:
# 資料看起來得像對應的資料類型
# a = int('3.234')
# a = int('3a234')
# a = float('3a5')
# print(a, type(a))

資料類型轉換應用

# 1 獲取資料
# 將輸入的資料轉換為整型,將整型結果賦值給變數number1
number1 = int(input('請輸入數位1:'))
number2 = float(input('請輸入數位2:'))

# 2 資料處理(運算)
# float = float - int
result = number2 - number1

# 3 輸出結果
# 字串的拼接: str + str
print('計算的結果是:' + str(result))  # 列印拼接後1個資料
print('計算的結果是:', str(result))   # 列印2個資料

資料類型轉換

  1. 轉換為整形: int(數據)
  2. 轉換為浮點型:float(數據)
  3. 轉換為字串:str(數據)
  4. 轉換為布林:bool(數據)

結果為False：bool(0)  bool(0.0)  bool(None)

  • 混合類型自動升級：
    1 + 2.14  返回的結果是 3.14
    1 + 3.0   返回結果是:  4.0
